## Uvod u Markdown

Naslovi
" ## " Heading h2

" ### " Heading h3

" #### " Heading h4


Uređivanje teksta
Bold tekst " ** ** "  ili " __ __ "

Italic tekst " * * " or " _ _ "

Precrtani tekst " ~~ ~~ "

Bold i italic " *** *** "


Citiranje
Za citiranje koristimo " > "


Naredba za citiranje koda:
git status
git add
git commit
