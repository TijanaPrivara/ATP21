# Akademsko i tehničko pisanje (ak. god. 2021./2022.)


## Uvod u Markdown

*Naslov*

" ## " Heading h2

" ### " Heading h3

" #### " Heading h4

___

*Uređivanje teksta*

Bold tekst " ** ** "  ili " __ __ "

Italic tekst " * * " or " _ _ "

Precrtani tekst " ~~ ~~ "

Bold i italic " *** *** "

___

*Citiranje*

Za citiranje koristimo " > "


*Naredba za citiranje koda:*

git status
git add
git commit
